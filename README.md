# 🌤️ SkyCast

A modern and responsive weather application that provides real-time weather information for cities around the world.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* ☁️ Current weather conditions
* 💧 Humidity information
* 🌬️ Wind speed details
* 📅 5-day weather forecast
* 📱 Fully responsive design
* 🌙 Modern dark-themed UI
* ⚡ Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* OpenWeatherMap API

## 📸 Preview

SkyCast offers a clean and modern interface for checking weather conditions quickly and efficiently.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/nandinikanchi006/skycast.git
```

2. Navigate to the project folder:

```bash
cd skycast
```

3. Open `index.html` in your browser.

## 🔑 API Setup

1. Create an account at OpenWeatherMap.
2. Generate an API key.
3. Replace the API key in the JavaScript file with your own key.
